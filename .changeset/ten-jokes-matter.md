Initial public release of `eslint-plugin-friday`, an opinionated set of 45 ESLint rules for consistent, LLM-friendly TypeScript and React code. The package ships as ESM and CommonJS with type declarations and exposes six flat-config presets: `base`, `base/recommended`, `react`, `react/recommended`, `all`, and `all/recommended`. The `/recommended` variants set their rules to error and the bare variants set them to warn.
